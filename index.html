Rock Paper Scissors Game

This game is currently played in the console.

Follow the instructions below to play this game:

  1. Open the Browser Console with CTRL + SHIFT + J
  2. Type game(); into the console and hit ENTER - a prompt window will appear at the top of your browser when the game begins
  3. You are playing against the computer. Be the first to 5 points and you win!
  4. Please give me feedback on the game and code so I can improve!
  5. ENJOY!
